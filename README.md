## Installation

NodeJS  is requiried to be installed previously.

Install steps

```sh
git clone https://github.com/gurodriguez/frontendtest
cd <toyourdirectory>
npm install
```

create an .env file inside your folder with the next value
```sh
vim .env
```
- REACT_APP_API_URL=<URL To your backend>

Run the code
```sh
npm start
```

Thats all...
